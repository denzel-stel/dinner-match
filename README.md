# Dinner match

## Installation guide (Android)

### General 
Requirements: 
- Docker 
- Installed Android emulator

1. Copy `.env.example` to `.env` 
2. Fill in the appropriate values.

### Running the server
1. Run `make start` 

### Running the app
1. `cd app`
2. `npx expo run:android`
